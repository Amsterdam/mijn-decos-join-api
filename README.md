# Mijn Decos Join API

## Introduction

A REST API which discloses the decos-join data

### Requirements

- Access to Rattic

### Local development

1. Clone repo and `cd` into
2. Create a virtual env and activate
3. Run `pip install -r decosjoin/requirements.txt`
4. Run `sh run-dev.sh`

### Testing

1. Clone repo
2. Create a virtual env and activate
3. Run `pip install -r decosjoin/requirements.txt`
4. `cd decosjoin`
5. Run `python -m unittest`
